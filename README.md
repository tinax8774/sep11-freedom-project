## Context
I am a student at [HSTAT](https://www.hstat.org/) in the [Software Engineering Program](https://hstatsep.github.io/). The "Freedom Project" for SEP11 is a year-long project all about making something using JavaScript along with a third-party JS tool.

For my project, I chose to independently study **Typescript** in order to help me make **my app**.

## Implications
##### PROS
* Speeds up decision process when deciding what to read and watch
* Gives the user a list of suggestions and make their choosing a bit easier
* Allows for the user to make a choice of what their books and mmovie title is based on

##### CONS
* Isn't really beneficial for those who don't enjoy reading or watching movies
* Everyone has a different preference of movies and books so not that really helpful for all people
